Some Questions And With Answer

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for interacting
with HTML, XML, and other structured documents.

How do you select an element from the DOM?

We can select DOM By ID, By class,By tag name,By query selector,By query selector all

What is event delegation in the context of the DOM, and why is it useful?

Event Delegation in the DOM refers to the technique of attaching an event listener to a
parent element instead of directly to each child element.

Accessing and Modifying Attributes("setAttribute()","getAttribute()")
